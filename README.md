# Phantom

## 介绍
本主题借鉴修改于HTML5UP的phantom主题，适配Gridea并支持以下功能：
- 移动端页面适配
- 社交信息与备案信息设置
- 谷歌统计
- 友链页面
- 相册页面
- 日常页面
- Katex公式与代码高亮
- Waline与Twikoo评论支持

## 安装
使用 git clone 本项目到 Gridea 站点源文件路径的 themes 目录内，或者下载最新的源代码解压到 Gridea 站点源文件路径的 themes 目录内，然后 重启 Gridea 即可。 
Gridea 站点源文件路径：可在 Gridea 客户端设置页面查看。
